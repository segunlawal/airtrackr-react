## Run Locally

Clone the project

```bash
  git clone https://github.com/segunlawal/airtrackr-react.git
```

Go to the project directory

```bash
  cd airtrackr-react
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
